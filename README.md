# CLI Project Management Tool

## Project Overview

This is a Python-based Command-Line Interface (CLI) application for managing users, projects, and tasks. It supports persistent data storage using JSON and demonstrates object-oriented design and modular architecture.

---

## Features

- Create and list users
- Assign projects to users
- Add tasks to projects
- Mark tasks as complete
- Persistent storage using JSON
- Clean CLI interface using argparse
- Styled output using Rich
